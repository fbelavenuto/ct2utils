ct2utils

ct2utils é um conjunto de utilitários para manipulação de arquivos com extensão 'ct2', um formato utilizado para armazenar programas e jogos contidos em fita cassete do microcomputador TK2000. Esse formato foi criado para utilização no emulador do TK2000.
Copyright (R) 2011- Fábio Belavenuto

Cada utilitário possui uma finalidade, as quais poderão ser vistas na ajuda específica do utilitário.

Índice

  • bin2ct2 - Gera um arquivo CT2 à partir de arquivos binários (.b)
  • ct22bin - Extrai arquivos binários (.b) de arquivos CT2
  • ct22wav - Cria um arquivo de áudio (WAV) à partir de um arquivo CT2 para ser reproduzido em um TK2000 real
  • wav2ct2 - Interpreta um arquivo de áudio (WAV) e tenta extrair os dados binários, gerando um arquivo CT2
